Thank you for contacting Cantas email service.

We can inform you that for now the system you are referring to is the MLFF (Multi Lane Free Flow) system, which is a toll payment system without having to stop the vehicle. MLFF technology is present in Indonesia and is the first technology to implement a contactless cashless toll payment system or pay tolls without stopping based on an application called the CANTAS application for all classes of vehicle types.

So you can register for cantas application with foreign number but you need to Make sure the phone number is registered in your WhatsApp because the OTP will be send to your whatsapp message.

Currently only one payment system is available the named is Digiroad and you also can using payment sistem by Qris from your current bank

You can top up your balance via Digiroad by using Bank BJB Mobile Banking (Digi Mobile by Bank BJB users only) here are the steps:

1. Login to DIGI MOBILE application

2. Select Top-Up e-Wallet Menu

3. Select Top-Up digiRoad on the Service List

4. Input the Phone Number registered on digiRoad (example: 628123456xxxxxxx)

5. Input the Top Up Amount

6. Input Digi Mobile M-PIN

And also You can top up your balance via Digiroad by using QRIS for bank account owners:

1. Select TOPUP BALANCE

2. Enter the Phone Number registered in digiRoad (example: 628123456xxxxxxx)

3. Input the top-up amount

4. Scan QRIS through the application you choose then complete the payment.

5. Click "PAID" button
