Regarding Mr Nurhamdi Fahmi question about how to reset Cantas account, If you mean reset account to create a new account because you want to change your username, you can delete your Cantas account with these steps:
1. From main menu, select "Account"
2. Select "Delete account"
3. Select "Delete my account"
4. Enter the OTP code sent to your Whatsapp

Please note that the process of deleting account will take 15 days. You can cancel this process anytime within those 15 days by logging in again. After deleting the account, information related to travel history, fine history, and other information previously recorded in your account will also be deleted. After that you can install Cantas application and regist your new account.

If you only want to reset your password in Cantas application because of forgot your password, you only need to reset your password and here's how to reset the password:
1. Select forgot password on the application home screen
2. Select and enter the mobile number or email registered in the Cantas application, then create a new password
4. Then you will receive a link to change your password sent to WhatsApp or email
5. Open the link which will automatically enter the Web view to create a new password
6. You will be asked to create a new password and confirm the new password. After that select create a new password.
