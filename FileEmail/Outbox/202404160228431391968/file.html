Very well, Mr Muhammad Alvin. To change your email, you can do the following:
1. Enter the ""Account"" menu
2. Select the ""Profile"" menu then the details of personal data will appear. Select ""Contact Details""
3. Select ""Change"" and enter the new email you want to use. 
4. Press the ""Next"" button and a pop-up message will appear asking whether the new email entered is correct or not. Press the ""OK"" button if it is correct.

And to change your mobile number, you can do the following:
1. Enter the ""Account"" menu
2. Select the ""Profile"" menu then the details of personal data will appear. Select ""Contact Details""
3. Select ""Change"" and enter the new cell phone number you want to use. 
4. Press the ""Next"" button and a pop-up message will appear asking whether the new cell phone number entered is correct or not. Press the ""OK"" button if it is correct.
5. The OTP code will be sent to the new mobile number you entered
