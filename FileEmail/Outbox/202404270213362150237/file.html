Well, we can inform you about the question, you can choose to top up the balance via Digiroad using QRIS for bank account users in Indonesia. Here's how: 
1. Select the “Payment Method” menu 
2. Press the “Top Up” button 
3. Press QRIS button then press “TOPUP BALANCE” button 3. 
4. Enter the Phone Number registered in digiRoad (example: 628123456xxxxxxx) 
5. Select the top-up amount and click “Submit” 5. 
6. Confirm the amount and phone number that you have entered then click “Continue”. 
7. A QR code will appear that can be scanned through the mobile banking application for payment transactions. 
8. Click “Already Paid” if successful, you will receive a successful payment notification.
