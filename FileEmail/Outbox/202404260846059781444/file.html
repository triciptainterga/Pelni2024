If there is a change in vehicle data but with the same previous license plate, then you can make changes in the Cantas application in the following ways:
 1. Enter the Cantas application
 2. Select Account
 3. Select vehicle
 4. Select the vehicle whose data will be changed
 5. Make changes to vehicle data and photos of the latest vehicle documents
 6. Select Save changes if the data is correct
 
However, if there is a change in the vehicle license plate, then you can directly add the vehicle because the license plate cannot be changed from the previous license plate in the application. here's how to add a vehicle:
 1. Select the “Account” menu
 2. Select the “Vehicle” menu
 3. Click the “Add Vehicle” button or the Plus sign
 4. Select the toll category that matches your vehicle
 5. Enter your vehicle information, then click “Next”
 6. Upload the front and back photos of your vehicle, then click “Next”
 7. Upload a photo of your vehicle's front and rear vehicle registration (STNK)
 8. Click “Finish”
