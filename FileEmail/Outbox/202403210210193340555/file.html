Regarding Mr. Antony Tambunan question about top up method in Cantas application, currently cannot be done by using Gopay. We would like to inform you for top up balance in Cantas application only available by using Digiroad, you can  add DigiRoad balance using Bank BJB mobile banking or add DigiRoad balance using QRIS.  You can top up your balance via Digiroad by using Bank BJB Mobile Banking (Digi Mobile by Bank BJB users only) you can follow these step:
1. Login to DIGI MOBILE application
2. Select Top-Up e-Wallet Menu
3. Select Top-Up digiRoad on the Service List
4. Input the Phone Number registered on digiRoad (example: 628123456xxxxxxx)
5. Input the Top Up Amount
6. Input Digi Mobile M-PIN


You also can top up your balance via Digiroad by using QRIS for bank account owners.
1. Select TOPUP BALANCE 
2. Enter the Phone Number registered in digiRoad (example: 628123456xxxxxxx)
3. Input the top-up amount
4. Scan QRIS through the application you choose then complete the payment.
5. Click "PAID" button 

in the future for other payments will be available on the Cantas application, please wait for the latest information on the Cantas application or on our website at roatex.co.id.
