Alright Mr. Reinhard , according your question about new payment method we could inform you that right now there is new payment method for toll road, it is CANTAS. CANTAS itself using MLFF system, MLFF stands for "Multi Lane Free Flow" which is a contactless toll payment system that can be done without having to stop at the toll gate. This system is implemented in Indonesia through the CANTAS app with GNSS technology which uses GPS on smartphones to track the journey of toll road users. Currently, the CANTAS application is still in the trial phase on the Bali toll road, and in the future it will gradually be implemented on all toll roads in Indonesia. 

For using Cantas application, kindly Mr. Reinhard follow theses steps below:
1. Make sure you have added your vehicle data to your "Vehicle" menu 
2. Make sure you have activated one of the payment methods available on the Cantas application 
3. On the home page of the application select the "Trip" menu 
4. Select the vehicle you will use. Make sure the vehicle you will use is the same as the one you selected in the application 
5. Click "Start Trip" button to start your trip with the Cantas application
6. Make sure you have clicked the "Start Trip" before entering the toll gate
7. Click "Stop Trip" button after you exit the toll gate. If you forget to click the "Stop Trip" , the system will automatically end the trip after 30 minutes while you're outside of the toll road
