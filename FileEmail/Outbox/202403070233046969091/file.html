Regarding your question about how to change your username and password, we can tell you that changing the username in the Cantas application is currently not possible. You can create a new Cantas account with another username that you want with another phone number that is connected to Whatsapp.

If you want to create an account with a new username but still use the previous phone number, you can delete the account with a processing time of 15 days for permanent deletion. After deleting the account, information related to travel history, fine history, and other information previously recorded in your account will also be deleted.

Related to how to change password, here are the steps:

1. Select forgot password on the application home screen
2. Select and enter the mobile number or email registered in the Cantas application, then create a new password
4. Then you will receive a link to change your password sent to WhatsApp or email
5. Open the link which will automatically enter the Web view to create a new password
6. You will be asked to create a new password and confirm the new password. After that select create a new password
