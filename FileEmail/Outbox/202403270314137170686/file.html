Sehubungan dengan email mengenai top up saldo di akun cantas, berikut beberapa alternatif cara yang dapat digunakan:

  1. Dapat memilih penambahan saldo melalui Digiroad menggunakan Mobile Banking Bank BJB (khusus pengguna DigI Mobile by Bank BJB) cara berikut:
  • Login ke aplikasi DIGI MOBILE
  •  Pilih Menu Top Up e-Wallet
  • Pilih Top/Up digiRoad pada Daftar Pelayanan
  • Input Nomor Handphone yang terdaftar pada digiRoad (contoh : 628123456xxxxxxx)
  • Masukkan Nominal Pengisian saldo
  • Masukkan M-PIN Digi Mobile

2.  Dapat memilih untuk mengisi saldo melalui Digiroad menggunakan QRIS untuk pengguna rekening bank di Indonesia. Berikut caranya:

  • Pilih menu "Metode Pembayaran"
  • Tekan tombol "Isi Ulang"
  • Tekan tombol QRIS kemudian tekan tombol "TOPUP SALDO"
  • Masukkan Nomor Telepon yang terdaftar di digiRoad (contoh: 628123456xxxxxxx)
  • Pilih nominal isi ulang saldo dan klik "Kirim"
  • Konfirmasi untuk nominal dan no. telepon yang Bapak/Ibu masukkan kemudian tekan "Lanjutkan"
  • Akan muncul QR code untuk di pindai melalui aplikasi mobile banking untuk transaksi pembayaran
  • Klik "Sudah Bayar", jika berhasil akan menerima notifikasi pembayaran sudah berhasil

Sebagai informasi bahwa proses menambahkan saldo dengan menggunakan QRIS Ibu akan dikenakan Rp 1,500 untuk biaya administrasi.
