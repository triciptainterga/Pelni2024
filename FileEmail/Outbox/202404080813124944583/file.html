Answering Mr. Ramavino's Email, If you want to top up the Digiroad balance using a bank then you can use top up through Qris.

By scanning the Qris provided using automatic bank transfer in the following way:
1. Select the "Payment Method" menu 
2. Press the "Top up" button 
3. Press QRIS button then press "TOPUP BALANCE" button 3. 
4. Enter the Phone Number registered in digiRoad (example: 628123456xxxxxxx) 
5. Select the top-up amount and click "Submit" 5.
6. Confirm the amount and phone number that you have entered then click "Continue".
7. A QR code will appear that you can scan through the mobile banking application for payment transactions. 
8. Click "Already Paid" 

If successful, you will receive a successful payment notification For information that the process of adding a balance using QRIS you will be charged Rp 1,500 for administrative fees.
