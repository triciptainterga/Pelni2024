For the inauguration of the Cantas application, it will be announced through the official media of PT Roatex Indonesia Toll System (RITS) and the official PUPR media. Please wait for the latest info from us.

To use the Cantas application, things that need to be considered:
 1. Make sure you have added your vehicle data on the "Vehicle" menu 
 2. Make sure you have activated one of the payment methods available in the Cantas application
 3. On the home page of the application select the "Trip" menu
 4. Select the vehicle to be used. Make sure the vehicle to be used is the same as the one selected in the application
 5. Press the "Start Trip" button to start the trip with the Cantas application
 6. Make sure the "Start Trip" button has been clicked before entering the toll gate so as not to be subject to fines
 7. Press the "Stop Trip" button when you exit the toll gate. If you forget to click the "Stop Trip" button, the system will stop the trip automatically after 2 hours out of the toll road.
