Dapat kami sampaikan bahwa pembayaran Cantas sebagai berikut : 
"Anda dapat memilih untuk mengisi saldo melalui Digiroad menggunakan QRIS untuk pengguna rekening bank di Indonesia.
1. Pilih menu ""Metode Pembayaran""
2. Tekan tombol ""Isi Ulang""
3. Tekan tombol QRIS kemudian tekan tombol ""TOPUP SALDO"" 
4. Masukkan Nomor Handphone yang terdaftar di digiRoad (contoh: 628123456xxxxxxx)
5. Pilih nominal isi ulang saldo dan klik ""Kirim""
6. Konfirmasi untuk nominal dan no.telepon yang anda masukkan kemudian tekan ""Lanjutkan""
7. Akan muncul QR Code yang bisa anda pindai melalui aplikasi mobile banking untuk transaksi pembayaran
8. Klik ""Sudah Bayar"", jika berhasil anda akan menerima notifikasi pembayaran sudah berhasil"
