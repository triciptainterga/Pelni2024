Mohon kesediaannya untuk melakukan pengisian ulang saldo melalui DigiRoad dengan menggunakan Mobile Banking Bank BJB dan juga QRIS dengan cara berikut:

Mobile Banking Bank BJB (khusus pengguna DigI Mobile by Bank BJB).

1. Login ke aplikasi DIGI MOBILE

2. Pilih Menu Top Up e-Wallet

3. Pilih Top Up DigiRoad pada Daftar Pelayanan

4. Input Nomor Handphone yang terdaftar pada DigiRoad (contoh : 628123456xxxxxxx)

5. Masukkan nominal isi ulang pengisian saldo

6. Masukkan M-PIN Digi Mobile

 

QRIS:

1. Pilih menu "Metode Pembayaran"

2. Tekan tombol "Isi Ulang"

3. Tekan tombol QRIS kemudian tekan tombol "TOPUP SALDO"

4. Masukkan Nomor Telepon yang terdaftar di DigiRoad (contoh: 628123456xxxxxxx)

5. Pilih nominal isi ulang saldo dan klik "Kirim"

6. Konfirmasi untuk nominal dan no. telepon yang Bapak masukkan kemudian tekan "Lanjutkan"

7. Akan muncul QR code yang bisa di pindai melalui aplikasi mobile banking atau dompet elektronik untuk transaksi pembayaran

8. Klik "Sudah Bayar" jika berhasil Bapak akan menerima notifikasi pembayaran sudah berhasil

Sebagai informasi bahwa proses menambahkan saldo dengan menggunakan QRIS Bapak akan dikenakan Rp1.500 untuk biaya administrasi
