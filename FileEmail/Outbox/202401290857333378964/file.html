Thank you for contacting Cantas email service.

Regarding your questions, we can suggest you to follow these steps to use the 'Share Vehicle' feature: 

1. From main menu, select "Invitation"
2. Select "Check my shared vehicle"
3. Select vehicle that you want to share with other users in "My vehicles"
4. Select "invite users"
5. Enter the username you want to invite and the expiration date for the invitation.
6. Click "Send invitation"

And in order to receive the 'Share Vehicle' invitation in Cantas app, you can follow these steps:

1. From main menu, select "Invitation"
2. Select "Check vehicle invitations"

Alternatively, you can use these steps:
1. From main menu, select "Account"
2. Select "Vehicles sharing invitations"
