Regarding your question about how to delete other vehicles, we can tell you that shareing vehicles can be deleted through the first account that shared the vehicle with other users.

The way to delete a vehicle is as follows:
1. Enter the "Account" menu
2. Then the "Vehicle" menu
3. Select the license plate of the vehicle you want to delete
4. Select "Delete vehicle"
5. Done
