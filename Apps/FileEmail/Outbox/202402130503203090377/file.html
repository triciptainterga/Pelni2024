Kami informasikan untuk pengisian saldo dapat dilakukan dua cara sebagai berikut :

Dapat memilih penambahan saldo melalui Digiroad menggunakan Mobile Banking Bank BJB (khusus pengguna DigI Mobile by Bank BJB).
1. Login ke aplikasi DIGI MOBILE
2. Pilih Menu Top Up e-Wallet
3. Pilih Top/Up digiRoad pada Daftar Pelayanan
4. Input Nomor Handphone yang terdaftar pada digiRoad (contoh : 628123456xxxxxxx)
5. Masukkan Nominal Pengisian saldo
6. Masukkan M-PIN Digi Mobile

Dapat memilih untuk mengisi saldo melalui Digiroad menggunakan QRIS untuk pengguna rekening bank di Indonesia.
 1. Pilih menu "Metode Pembayaran"
 2. Tekan tombol "Isi Ulang"
 3. Tekan tombol QRIS kemudian tekan tombol "TOPUP SALDO" 
 4. Masukkan Nomor Handphone yang terdaftar di digiRoad (contoh: 628123456xxxxxxx)
 5. Pilih nominal isi ulang saldo dan klik "Kirim"
 6. Konfirmasi untuk nominal dan no.telepon yang terdaftar kemudian tekan "Lanjutkan"
 7. Akan muncul QR Code yang bisa pindai melalui aplikasi mobile banking untuk transaksi pembayaran
 8. Klik "Sudah Bayar", jika berhasil akan menerima notifikasi pembayaran sudah berhasil

Terkait hal diatas, jika kami tidak menerima tanggapan atau informasi dari Bapak/Ibu dalam waktu 2X24 jam maka kami anggap Bapak/Ibu sudah memahami masalah ini dan permasalahan ini akan kami tutup.
