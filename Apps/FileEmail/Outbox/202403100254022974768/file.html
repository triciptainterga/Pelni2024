MLFF stands for "Multi Lane Free Flow" which is a contactless toll payment system that can be done without having to stop at the toll gate. This system is implemented in Indonesia through the CANTAS app with GNSS technology which uses GPS on smartphones to track the journey of toll road users. Currently the CANTAS application is still in the trial phase on the Bali toll road, and in the future it will gradually be implemented on all toll roads in Indonesia.
