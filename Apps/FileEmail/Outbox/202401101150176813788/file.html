Menjawab Email Ibu Adinda, untuk pembayaran saat ini di aplikasi tersedia menggunakan digiRoad. Perihal pembayaran metode lain saat ini masih belum tersedia namun untuk pembayaran bisa dibantu dengan melakukan top up pada penyedia layanan pembayaran digiRoad. Berikut cara melakukan top up atau pengisian saldo digiRoad:

Ibu dapat memilih penambahan saldo melalui Digiroad menggunakan Mobile Banking Bank BJB (khusus pengguna DigI Mobile by Bank BJB).
1. Login ke aplikasi DIGI MOBILE
2. Pilih Menu Top Up e-Wallet
3. Pilih Top/Up digiRoad pada Daftar Pelayanan
4. Input Nomor Handphone yang terdaftar pada digiRoad (contoh : 628123456xxxxxxx)
5. Masukkan Nominal Pengisian saldo
6. Masukkan M-PIN Digi Mobile"

Kemudian Ibu dapat memilih untuk mengisi saldo melalui Digiroad menggunakan QRIS untuk pengguna rekening bank di Indonesia.
1. Pilih menu ""Metode Pembayaran""
2. Tekan tombol ""Isi Ulang""
3. Tekan tombol QRIS kemudian tekan tombol ""TOPUP SALDO"" 
4. Masukkan Nomor Handphone yang terdaftar di digiRoad (contoh: 628123456xxxxxxx)
5. Pilih nominal isi ulang saldo dan klik ""Kirim""
6. Konfirmasi untuk nominal dan no.telepon yang Bapak/Ibu masukkan kemudian tekan ""Lanjutkan""
7. Akan muncul QR Code yang bisa Bapak/Ibu pindai melalui aplikasi mobile banking untuk transaksi pembayaran
8. Klik ""Sudah Bayar"", jika berhasil Bapak/Ibu akan menerima notifikasi pembayaran sudah berhasil

Untuk kedepannya kami akan menambaahkan metode pembayaran lainnya yang akan segera tersedia di Aplikasi CANTAS. Tunggu perkembangan kami ya Bu.
