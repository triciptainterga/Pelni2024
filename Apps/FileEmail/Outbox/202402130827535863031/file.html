We would like to Inform Mr. Jauhar Andrewicaksono how to share your vehicle in your Cantas application. You can send "Share vehicle" invitation with other users with these steps:

1. From main menu, select "Invitation"

2. Select "Check my shared vehicle"

3. Select vehicle that you want to share with other users in "My vehicles"

4. Select "invite users"

5. Enter the username you want to invite and the expiration date for the invitation.

6. Click "Send invitation"

 

Make sure you invited friends click accept in Cantas Application, your friend’s can check or accept "Share vehicle" invitation from other you with these steps:
1. From main menu, select "Invitation"
2. Select "Check vehicle invitations"

or you friend can use these steps instead:
1. From main menu, select "Account"
2. Select "Vehicles sharing invitations"
