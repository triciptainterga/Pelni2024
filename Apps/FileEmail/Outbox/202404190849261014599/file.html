Thank you for contacting Cantas email service.

We can inform you that for now the system you are referring to is the MLFF (Multi Lane Free Flow) system, which is a toll payment system without having to stop the vehicle. MLFF technology is presented in Indonesia and is the first technology to implement a contactless cashless toll payment system or pay tolls without stopping based on an application called the CANTAS application for all classes of vehicle types. to use the Cantas application things that need to be considered:

1. Make sure you have added your vehicle data to your "Vehicle" menu

2. Make sure you have activated one of the payment methods available on the Cantas application

3. On the home page of the application select the "Trip" menu

4. Select the vehicle you will use. Make sure the vehicle you will use is the same as the one you selected in the application

5. Click "Start Trip" button to start your trip with the Cantas application.

6. Make sure you have clicked the "Start Trip" before entering the toll gate.

7. Click "Stop Trip" button after you exit the toll gate. If you forget to click the "Stop Trip" , the system will automatically end the trip after 12 hours while you're outside of the toll road.

But, if you rent a vehicle, you can coordinate with the vehicle owner or rental so that it can be ensured that the vehicle you rent is shared by the rental vehicle owner and you are not required to register the vehicle yourself to avoid violations or fines caused by the negligence of other drivers using the same vehicle. 
All informationwill be on the website. But, the official Cantas website is not currently available to access. Once it is ready, it will be announced through the official channels of PT. Roatex Indonesia Toll System (RITS) and the official channels of the Ministry of Public Works and Public Housing (PUPR). We appreciate your patience for the latest updates from us. Also, we can inform you that for now in the transition period E-toll and Flo can still be used when you use the toll road. Regarding the upcoming E-toll and Flo will be replaced with the CANTAS Application as a payment method in using toll roads throughout Indonesia.
