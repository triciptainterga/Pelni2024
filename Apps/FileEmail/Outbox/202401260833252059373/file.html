The new payment system in Bali Toll road will be called MLFF. MLFF stands for "Multi Lane Free Flow" which is a non-stop toll payment process, where the toll road users do not need to stop their vehicles at the toll gate to make payments. You need to download and install CANTAS in your smartphone for using MLFF based toll road system. CANTAS is a smart application designed with (MLFF) or Multi Lane Free Flow system that utilize  (GNSS) or Global Navigation Satellite System technology  which allows vehicle users to make payments just by using their device without having to stop.

To use the CANTAS application things that need to be considered:
1. Make sure you have added your vehicle data to your ""Vehicle"" menu
2. Make sure you have activated one of the payment methods available on the CANTAS application
3. On the home page of the application select the ""Trip"" menu
4. Select the vehicle you will use. Make sure the vehicle you will use is the same as the one you selected in the application
5. Click ""Start Trip"" button to start your trip with the CANTAS application.
6. Make sure you have clicked the ""Start Trip"" before entering the toll gate.
7. Click ""Stop Trip"" button after you exit the toll gate. If you forget to click the ""Stop Trip"" , the system will automatically end the trip after 2 hours while you're outside of the toll road.

For now, MLFF still on trial and transition period for invited users. E-money and Flo can still be used when use your rental car on the toll road by choosing regular card tapping toll road gate just before get into the toll road. Regarding the upcoming E-toll and Flo will be replaced with the CANTAS Application as a payment method in using toll roads throughout Indonesia, this is still awaiting confirmation from related parties which will be announced through the official media of PT Roatex Indonesia Toll System (RITS) and official PUPR media. You can check their official media regularly for updated information about CANTAS later.
