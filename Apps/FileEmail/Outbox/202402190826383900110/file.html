MLFF technology is present in Indonesia and is the first technology to implement a contactless cashless toll payment system or pay tolls without stopping based on an application called the CANTAS application for all classes of vehicle types. The MLFF system uses Global Navigation Satellite System (GNSS) digital technology that allows toll road users' trips to be known through GPS on smart phones.
