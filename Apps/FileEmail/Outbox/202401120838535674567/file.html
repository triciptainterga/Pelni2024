Thank you for contacting Cantas email service.

we may informed that 

1. to use the Cantas application things that need to be considered:
- Make sure you have added your vehicle data to your "Vehicle" menu
- Make sure you have activated one of the payment methods available on the Cantas application
- On the home page of the application select the "Trip" menu
- Select the vehicle you will use. Make sure the vehicle you will use is the same as the one you selected in the application
- Click "Start Trip" button to start your trip with the Cantas application.
- Make sure you have clicked the "Start Trip" before entering the toll gate.
- Click "Stop Trip" button after you exit the toll gate. If you forget to click the "Stop Trip" , the system will automatically end the trip after 2 hours while you're outside of the toll road.

2. we can inform you that currently there is no limit for license plates that can be registered in 1 account.

3. if your family want use your car, you can use "share vehicle" in cantas application

4. for currently, toll roads that can already be used with the cantas application are Nusa Dua – Ngurah Rai – Benoa (Bali)
