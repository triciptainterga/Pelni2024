Cantas is a mobile application to implement a cashless toll payment transaction system carried out at normal speed using contactless technology or known as Multi Lane Free Flow (MLFF).

Cantas uses Global Navigation Satellite System (GNSS) technology. GNSS technology is a navigation satellite system that determines positions with certain coordinates accurately, continuously and simultaneously without being disturbed by weather.

how CANTAS works is: 

  • Toll road users first download the Cantas Application on the Play Store and App Store
  • Register by entering personal data and vehicle data
  • Select and connect with sources of payment (source of funds) such as electronic money, direct debit banking, etc.
  • Ensure sufficient funds
  • Ensure the application is active
  • The system in the application will be detected when crossing the toll road, then the user's electronic balance is automatically deducted (transaction process).
