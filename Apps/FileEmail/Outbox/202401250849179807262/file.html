Thank you for contacting Cantas email service.

We can inform you that the system you are referring to is the MLFF system. MLFF stands for "Multi Lane Free Flow" which is a non-stop toll payment process, where toll road users do not need to stop their vehicles at the toll gate to make payments.

MLFF system can be used through an application system called CANTAS. Cantas is a mobile application to implement a cashless toll payment transaction system carried out at normal speed using contactless technology or known as Multi Lane Free Flow (MLFF).

Here are the steps to use the CANTAS application: 

- Toll road users first download the Cantas Application on the Play Store and App Store
- Register by entering personal data and vehicle data
- Select and connect with sources of payment (source of funds) such as electronic money, direct debit banking, etc.
- Ensure sufficient funds
- Ensure the application is active
- The system in the application will be detected when crossing the toll road, then the user's electronic balance is automatically deducted (transaction process).

As I mentioned earlier, you must register your vehicle whether it is a private car or a rental car, therefore the vehicle you are using is registered and not subject to fines when using the toll road.
