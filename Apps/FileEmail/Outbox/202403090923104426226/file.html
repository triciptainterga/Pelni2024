Terkait pertanyaan Bapak Firmane Ben untuk cara pengisian saldo di aplikasi Cantas pastikan Bapak sudah mengunduh aplikasi Digiroad. Bapak dapat melakukan pembayaran dengan menggunakan Digiroad menggunakan Mobile Banking Bank BJB dan Digiroad menggunakan Qris. Berikut cara menambahkan saldo melalui Digiroad menggunakan Mobile Banking Bank BJB (khusus pengguna DigI Mobile by Bank BJB). 

1. Login ke aplikasi DIGI MOBILE
2. Pilih Menu Top Up e-Wallet
3. Pilih Top/Up digiRoad pada Daftar Pelayanan
4. Input Nomor Handphone yang terdaftar pada digiRoad (contoh : 628123456xxxxxxx)
5. Masukkan Nominal Pengisian saldo
6. Masukkan M-PIN Digi Mobile"

 

Bapak juga dapat memilih untuk mengisi saldo melalui Digiroad menggunakan QRIS untuk pengguna rekening bank di Indonesia. Berikut caranya:

1. Pilih menu "Metode Pembayaran"
2. Tekan tombol "Isi Ulang"
3. Tekan tombol QRIS kemudian tekan tombol "TOPUP SALDO"
4. Masukkan Nomor Telepon yang terdaftar di digiRoad (contoh: 628123456xxxxxxx)
5. Pilih nominal isi ulang saldo dan klik ""Kirim"
6. Konfirmasi untuk nominal dan no. telepon yang Bapak/Ibu masukkan kemudian tekan ""Lanjutkan""
7. Akan muncul QR code yang bisa Bapak/Ibu pindai melalui aplikasi mobile banking untuk transaksi pembayuran
8. Klik ""Sudah Bayar"", jika berhasil Bapak/Ibu akan menerima notifikasi pembayaran sudah berhasil
