Kami memahami pertanyaan Bapak mengenai cara top up saldo DigiRoad menggunakan aplikasi DANA. Dapat kami sampaikan bahwa saat ini belum tersedia opsi top up saldo menggunakan layanan tersebut. Namun Bapak tetap dapat top up menggunakan QRIS apabila tidak memiliki rekening Bank BJB. Berikut cara top up saldo DigiRoad menggunakan QRIS.

1. Pilih menu "Metode Pembayaran"

2. Tekan tombol "Isi Ulang"

3. Tekan tombol QRIS kemudian tekan tombol "TOPUP SALDO"

4. Masukkan Nomor Telepon yang terdaftar di digiRoad (contoh: 628123456xxxxxxx)

5. Pilih nominal isi ulang saldo dan klik "Kirim"

6. Konfirmasi untuk nominal dan no. telepon yang Bapak/Ibu masukkan kemudian tekan "Lanjutkan"

7. Akan muncul QR code yang bisa Bapak/Ibu pindai melalui aplikasi mobile banking untuk transaksi pembayaran

8. Klik "Sudah Bayar", jika berhasil Bapak/Ibu akan menerima notifikasi pembayaran sudah berhasil

 

Sebagai informasi bahwa proses menambahkan saldo dengan menggunakan QRIS Bapak akan dikenakan Rp 1,500 untuk biaya administrasi.
