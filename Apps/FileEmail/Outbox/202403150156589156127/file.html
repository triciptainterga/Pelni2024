Acording to your question about Cantas, we would like to inform you what is Cantas. Cantas is an application designed with (MLFF) or Multi Lane Free Flow system that utilize (GNSS) or Global Navigation Satellite System technology which allows vehicle users to make payments just by using their device without having to stop. MLFF itself stands for "Multi Lane Free Flow" which is a contactless toll payment system that can be done without having to stop at the toll gate. This system is implemented in Indonesia through the CANTAS app with GNSS technology which uses GPS on smartphones to track the journey of toll road users.
