Berikut kami informasikan cara top up saldo Digiroad menggunakan QRIS untuk pengguna rekening bank selain BJB di Indonesia:
1. Pilih menu "Metode Pembayaran"
2. Tekan tombol "Isi Ulang"
3. Tekan tombol QRIS kemudian tekan tombol "TOPUP SALDO" 
4. Masukkan Nomor Telepon yang terdaftar di digiRoad (contoh: 628123456xxxxxxx) 
5. Pilih nominal isi ulang saldo dan klik "Kirim"
6. Konfirmasi untuk nominal dan nomor telepon kemudian tekan "Lanjutkan" 
7. Akan muncul QR code yang bisa di pindai melalui aplikasi mobile banking untuk transaksi pembayaran 
8. Klik "Sudah Bayar" jika telah berhasil dan notifikasi pembayaran akan diterima.

Sebagai informasi tambahan, top up saldo dengan menggunakan QRIS akan dikenakan Rp 1,500 untuk biaya administrasi.
