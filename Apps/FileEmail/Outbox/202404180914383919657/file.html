Thank you for contacting Cantas email service.

Regarding to your question, we gladly inform you that The MLFF System operates by detecting either the E-OBU device or the CANTAS app installed on the customer's smartphone. It uses GNSS to pinpoint the vehicle's location and cameras on the Gantry to read the license plate and vehicle type, ensuring it matches the customer's registered information. Using the MLFF system simplifies payment processing and reduces congestion at Toll gates, eliminating the need for electronic card tapping previously required.

If you have a car rental business, we recommend you to use OBU in Cantas. but for now OBU is not yet available and please wait for the release.

To receive full information you can visit official media of PT Roatex Indonesia Toll Sistem (RITS) and the official Ministry of Public Works and Housing (PUPR) media. for informatiion that website not yet released and please take the time to wait for the latest info from us.

If you want to keep using the e-toll card, it can still be used during the transition period and we inform you that there is no problem if you still using e-toll card.
